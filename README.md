# ContextSync Collaboration Repo

Shared team brain for ContextSync: contracts, decisions, identity, and the relationship graph.

## Structure

- `identity/` — team roles and AI boundaries
- `contracts/` — API/schema/interface definitions
- `decisions/` — architecture decision records
- `playbooks/` — team workflow rules
- `skills/` — approved AI capabilities
- `graph/` — auto-maintained relationship graph

Clone this repo alongside your product repo and point ContextSync at both paths.
